OData to DocumentDB SQL Translator

This library provides methods to convert OData queries into SQL statements in C#.

Installation

The nuget package can be found at https://www.nuget.org/packages/Microsoft.Azure.Documents.OData.Sql

To install install this package in Visual Studio, run the following command in the Package Manager Console:

Install-Package Microsoft.Azure.Documents.OData.Sql
